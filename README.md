# CyberscanX

CyberscanX is an educational SQL injection scanner with a Streamlit UI.

### ⚠ Disclaimer
This tool must be used ONLY on legal, intentionally vulnerable demo servers such as:
- DVWA  
- JuiceShop  
- WebGoat  
- Mutillidae  
- bWAPP  

Do NOT scan public websites.
